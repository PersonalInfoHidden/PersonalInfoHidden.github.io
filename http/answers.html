Ord Definition
HTML Hypertext Markup Language
Storyboards Kommunikation och styling
Wireframes Testar funktionalitet, t.ex. med "Figma"
Klient Skickar HTTP-request för resurs
HTTP Request med HEAD Method Skickas utan body, bara header
Hallway Testing Slumpmässiga personer testar sidan
Webteknologier
  1. HTML - Textidentifiering med (h2) taggar
  2. CSS - Gör (h2) taggar blå
  3. JS - Validerar klientsidekod
  4. Ruby - Anpassad backend på serversidan
FrontEnd och Backend FrontEnd på klientsidan, Backend på serversidan
FTP-verktyg Flyttar filer mellan datorer via nätverk

Svar:

  1. Storyboards används både för enklare kommunikation mellan utvecklaren och klienten men också för att styla.
  2. Wireframes används för att testa funktionalitet som med prototype verktyget "Figma".
  3. När en Klient frågar om en resurs skickar den en http/https request till en api endpoint med en header som inkluderar ett http GET method.
  4. En http request med en HEAD method skickas till en endpoint kan responsen inte inkludera en body utan endast en header.
  5. Hallway testing utförs genom att man har slumpmässiga individer som testar sidan
    1. Grafisk designer - Storyboards
    2. FrontEnd programmerare - HTML och CSS
    3. Användarupplevelsedesigner - "Look and feel"
    4. Backend-programmerare - PHP-skript
  7. W3C skapar webstandards över hela världen.
    1. HTML - Indentifierar text som (h2) tag
    2. CSS - Gör alla (h2) taggar blå
    3. JS - Validerar kod på klientsidan
    4. Ruby - Skapar en anpassan backend på serversidan
  9. FrontEnd arbetar på klientsidan och backend på serversidan.
  10. Ett FTP-verktyg används för att flytta filer mellan datorer via nätverk. Du installerar en FTP-klient, som FileZilla, ansluter till servern och börjar överföra filer.
